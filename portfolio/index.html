Procedural Ivy Generator HDA

This project demonstrates the generation of a Houdini Digital Asset for ivy growth. This procedural tool can generate a variety of different scenes based on user defined parameter values. The ivy is generated using a sort of particle based method first introduced in 1985 by Reeves and Blau [1].

Escher inspired Penrose staircase with Bouncing Ball

The aim of the project was to model a Penrose (impossible) staircase inspired by M.C Escher’s Assending and Decending artwork and have a simple bouncing ball interacting with it.

Growth Simlutaions

The video here demonstrates several different growth algorithms implemented in Houdini. Each are done using the VEX programming language.

Artist Friendly Garden Generator HDA

This garden generator Houdini Digital Asset was created during the course of my Masters degree at Bournemouth University. It was my first introduction to Houdini, and was created in line with the assignment brief.

Creature Rig

This rig was created for a group project during the MSc Computer Animation and Visual Effects. Having little to no experience rigging prior to this creature, I began to teach myself the fundamentals and necessary requirements to complete a game-ready rig.

RenderMan Lemon

As part of a Simulation and Rendering assignment during the course of my Master’s degree, this lemon was completed using RenderMan and it’s associated API. It was modeled directly in RIB and shaded using OSL (Open Shading Language).
